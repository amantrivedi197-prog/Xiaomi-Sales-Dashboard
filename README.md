# 📱 Xiaomi Sales Dashboard - Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze Xiaomi smartphone sales data. It provides a clear overview of sales performance, customer purchasing behavior, payment methods, and brand-wise performance through interactive visualizations.

## 🎯 Objectives
- Analyze overall sales performance.
- Compare sales across different mobile brands and models.
- Identify monthly sales trends.
- Understand customer payment preferences.
- Visualize city-wise sales distribution.
- Monitor customer ratings and product performance.

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Excel

## 📊 Dashboard Highlights
- 💰 Total Sales: *769M*
- 📦 Total Quantity Sold: *19K*
- 🧾 Total Transactions: *4K*
- 💵 Average Sales: *40K*

## 📈 Visualizations Included
- Monthly Sales Trend
- Sales by Brand
- Sales by Mobile Model
- City-wise Sales Map
- Payment Method Distribution
- Customer Rating Analysis
- Interactive Filters (Month, Brand, Mobile Model, City, Payment Method)
